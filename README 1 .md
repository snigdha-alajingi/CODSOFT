# Iris Flower Classification

This project uses Machine Learning to classify Iris flowers into different species based on their features such as sepal length, sepal width, petal length, and petal width.

## About the Project

The Iris dataset is one of the most popular beginner datasets in Machine Learning.  
The goal of this project is to train a model that can accurately predict the species of an Iris flower.

The species classified are:
- Iris Setosa
- Iris Versicolor
- Iris Virginica

# Titanic Survival Prediction

This project predicts whether a passenger survived the Titanic disaster using Machine Learning algorithms.

## About the Project

The Titanic dataset is one of the most famous datasets for beginners in Data Science and Machine Learning.

The goal is to analyze passenger data and build a model that predicts survival.

## Features Used

- Age
- Gender
- Passenger Class
- Fare
- SibSp
- Parch

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow

1. Data Collection
2. Data Cleaning
3. Exploratory Data Analysis
4. Feature Engineering
5. Model Training
6. Prediction and Evaluation

## Machine Learning Algorithms

- Logistic Regression
- Decision Tree
- Random Forest

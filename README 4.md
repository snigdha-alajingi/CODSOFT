# Credit Card Fraud Detection

This project uses Machine Learning techniques to detect fraudulent credit card transactions.

## About the Project

Credit card fraud detection is an important real-world application of Machine Learning.

The aim of this project is to identify fraudulent transactions from transaction data using classification algorithms.

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Workflow

1. Data Collection
2. Data Preprocessing
3. Handling Imbalanced Data
4. Exploratory Data Analysis
5. Model Training
6. Fraud Detection

## Features

- Data visualization
- Fraud analysis
- Classification models
- Performance evaluation

## Machine Learning Algorithms

- Logistic Regression
- Random Forest
- Decision Tree
